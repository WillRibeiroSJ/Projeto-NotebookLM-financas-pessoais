# Guia Prático para Organização e Investimentos — NotebookLM

Este projeto utiliza o NotebookLM como ferramenta de Inteligência Artificial Generativa aplicada à educação financeira pessoal. O objetivo é organizar, sintetizar e explicar de forma prática como distribuir a renda entre despesas, reserva de emergência e investimentos iniciais, auxiliando na tomada de decisões financeiras conscientes. Todo o conteúdo é baseado exclusivamente nas fontes adicionadas ao NotebookLM, garantindo consistência e confiabilidade. O NotebookLM do projeto pode ser acessado no link abaixo.

https://notebooklm.google.com/notebook/63371a4d-9ef1-4cd6-bc31-297d3b5e2931

---

## Objetivo do Projeto
Criar um sistema de IA que funcione como um **guia educacional**, capaz de:
- Organizar e sintetizar conceitos fundamentais de finanças pessoais usando IA generativa;
- Demonstrar como distribuir a renda entre despesas, reserva de emergência e investimentos iniciais;
- Criar um especialista virtual focado em planejamento financeiro e tomada de decisões conscientes.

---

## Funcionalidades do NotebookLM
- Resumos em áudio;
- Resumo em vídeo;
- Mapa mental;
- Relatórios explicativos;
- Cartão didático;
- Testes conceituais;
- Infográficos.
